### Question 1
^ae0451c7-9260-4d9b-81ea-cbbee407a038

#### Question

````

```c
#include <stdio.h>

int main() {
    int x = 42;
    printf("The answer is %d\n", x);
    return 0;
}
```
````

#### Answer

````
The Rust Equivalent is given by:

```rust
fn main() {
    let x = 42;
    println!("The answer is {}", x);
}
```
````
